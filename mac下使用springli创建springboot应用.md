### 环境准备
生成springboot有以下几种方式
1. maven
2. gradle
3. springli
4. spring官网可以点击生成脚手架

spring官网介绍说springli是最快的方式，所以我使用了springli来构建。

mac安装springli我使用的是`brew`安装，如果没使用过`brew`请移步[brew](http://brew.sh/).

brew安装springli
```
$ brew tap pivotal/tap
$ brew install springboot
```
